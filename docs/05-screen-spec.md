# 05. 画面設計

## 画面一覧

| 画面ID | 画面名 | 用途 |
|---|---|---|
| S01 | 初回接続画面 | Copilot 接続前の案内と接続開始 |
| S02 | メイン画面 | 新しい相談の開始、必要時モード / 常時モード利用 |
| S04 | 会話画面 | AI との継続対話、回答の保存、常時モード状態表示 |
| S05 | ナレッジ管理画面 | 個人ナレッジの閲覧・編集・削除・エクスポート |
| S06 | 設定画面 | 初期モード、頻度、除外 glob の変更 |
| S07 | 制限・エラー画面 | 接続失敗や利用制限の案内 |
| S08 | 相談履歴画面 | 過去の相談一覧と再開 |

補足: 初期案にあった `S03 アドバイス詳細画面` と `S04 送信範囲確認画面` は、現行 v1 仕様では採用しない。  
現在の S04 は会話画面として扱う。

## 画面項目定義

### 5.1 S01 初回接続画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S01-01 | 画面タイトル | NaviCom の見出し |
| S01-02 | 概要説明 | 学習支援目的であることの説明 |
| S01-03 | 機能概要 | 文脈付き相談、会話継続、履歴、ナレッジ保存の案内 |
| S01-04 | 接続状態 / メッセージ | 未接続 / 接続中 / 利用不可などの補助表示 |
| S01-05 | Copilotに接続 | 接続開始ボタン |

### 5.2 S02 メイン画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S02-01 | 接続状態表示 | 接続済み / 制限中 / 未接続 |
| S02-02 | モード切替 | 必要時モード / 常時モード |
| S02-03 | 入力欄 | 最初の質問入力 |
| S02-04 | 送信ボタン | 新しい相談として送信 |
| S02-05 | 選択範囲チップ | 現在の選択範囲がある場合の簡易表示 |
| S02-06 | 常時モード状態 | アイドル待ち、クールダウン、一時停止状態 |
| S02-07 | 常時モード一時停止 | 常時助言の一時停止 / 再開 |
| S02-08 | 会話履歴 | S08 へ遷移 |
| S02-09 | ナレッジ管理 | S05 へ遷移 |
| S02-10 | 設定 | S06 へ遷移 |
| S02-11 | ステータスメッセージ | 接続やリクエスト失敗時の補助表示 |

専用の `深掘り` ボタン、`詳細を見る` ボタン、`送信範囲確認` ボタンは現行 v1 仕様では必須要件に含めない。

### 5.3 S04 会話画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S04-01 | 会話タイトル | 現在の相談タイトル |
| S04-02 | 戻る | S02 へ戻る |
| S04-03 | 接続状態表示 | 接続済み / 制限中 / 未接続 |
| S04-04 | 会話履歴 | ユーザー発話と AI 応答の一覧 |
| S04-05 | 入力欄 | 続きの質問入力 |
| S04-06 | 送信ボタン | 継続相談の送信 |
| S04-07 | 選択範囲チップ | 現在の選択範囲がある場合の簡易表示 |
| S04-08 | モード切替 | 必要時モード / 常時モード |
| S04-09 | 常時モード状態 | アイドル待ち、クールダウン、一時停止状態 |
| S04-10 | 常時モード一時停止 | 常時助言の一時停止 / 再開 |
| S04-11 | 回答アクション | 回答のコピー、ナレッジ保存 |
| S04-12 | 新しい相談 | 新規会話の作成 |
| S04-13 | 会話履歴 | S08 へ遷移 |
| S04-14 | ナレッジ管理 | S05 へ遷移 |
| S04-15 | 設定 | S06 へ遷移 |

### 5.4 S05 ナレッジ管理画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S05-01 | 検索欄 | ナレッジ検索 |
| S05-02 | 状態フィルタ | 有効 / 無効 / すべて |
| S05-03 | ナレッジ一覧 | 要約、状態、更新日時の表示 |
| S05-04 | ナレッジ詳細 | 本文やメモの表示 |
| S05-05 | 編集 | ナレッジ編集 |
| S05-06 | 無効化 | 再利用対象から外す |
| S05-07 | 削除 | ナレッジ削除 |
| S05-08 | エクスポート | JSON または Markdown 出力 |
| S05-09 | リセット | 全件リセット |
| S05-10 | 戻る | 元画面へ戻る |

### 5.5 S06 設定画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S06-01 | 初期モード | 起動時または接続後に使用するモード |
| S06-02 | リクエスト間隔 | 自動助言の最小間隔 |
| S06-03 | アイドル時間 | 自動助言前の必要無操作時間 |
| S06-04 | 保護済み除外パターン | 常に除外される glob の表示 |
| S06-05 | 追加除外 glob | ユーザー指定除外パターン |
| S06-06 | 保存 | 設定保存 |
| S06-07 | 変更を元に戻す | 未保存の変更を破棄 |
| S06-08 | 初期値に戻す | 設定初期化 |
| S06-09 | 戻る | 元画面へ戻る |

初期案にあった、重複抑制、送信対象カテゴリ ON/OFF、大容量閾値、ナレッジ再利用 ON/OFF、送信前確認 ON/OFF は、現行 v1 仕様では設定画面の必須項目に含めない。

### 5.6 S07 制限・エラー画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S07-01 | 状態タイトル | 同意未取得などの状態名 |
| S07-02 | 状態説明 | 問題の説明 |
| S07-03 | 推奨アクション | 次に行うべき操作 |
| S07-04 | 再試行 | 接続 / 送信再試行 |
| S07-05 | Copilotセットアップ案内 | Copilot 導入案内 |
| S07-06 | Workspace信頼案内 | Workspace Trust への誘導 |
| S07-07 | 戻る | 元画面または S01 へ戻る |

### 5.7 S08 相談履歴画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S08-01 | 相談一覧 | 過去の相談タイトル、更新日時の表示 |
| S08-02 | 相談を開く | 選択した相談を S04 で再開 |
| S08-03 | 新規相談 | 新しい相談の作成 |
| S08-04 | 履歴削除 | 相談履歴の削除 |
| S08-05 | 戻る | S02 へ戻る |
