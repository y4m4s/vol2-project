# 05. 画面設計

## 画面一覧

| 画面ID | 画面名 | 用途 |
|---|---|---|
| S01 | 初回接続画面 | Copilot 接続前の案内と接続開始 |
| S02 | メイン画面 | AI との対話、必要時モード / 常時モード利用 |
| S03 | アドバイス詳細画面 | 助言と根拠の詳細表示 |
| S04 | 送信範囲確認画面 | AI に送られる情報と除外情報の確認 |
| S05 | ナレッジ管理画面 | 個人ナレッジの閲覧・編集・削除・エクスポート |
| S06 | 設定画面 | モード、送信範囲、除外、ナレッジ設定の変更 |
| S07 | 制限・エラー画面 | 接続失敗や利用制限の案内 |

## 画面項目定義

### 5.1 S01 初回接続画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S01-01 | 画面タイトル | 画面の見出し |
| S01-02 | 概要説明 | 学習支援目的であることの説明 |
| S01-03 | 接続状態表示 | 未接続 / 接続中 / 同意待ち / 利用不可 |
| S01-04 | 送信範囲要約 | 送信対象の概要 |
| S01-05 | 送信除外要約 | 除外対象の概要 |
| S01-06 | Copilotに接続 | 接続開始ボタン |
| S01-07 | 送信範囲を確認 | S04 へ遷移 |
| S01-08 | 設定を開く | S06 へ遷移 |
| S01-09 | エラーメッセージ領域 | 接続失敗理由の表示 |
| S01-10 | 利用条件案内 | Copilot 利用、ネット接続、Trusted Workspace の案内 |

### 5.2 S02 メイン画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S02-01 | 接続状態表示 | 接続済み / 制限中 |
| S02-02 | モード切替 | 必要時モード / 常時モード |
| S02-03 | 会話履歴 | ユーザー発話と AI 応答の一覧 |
| S02-04 | 入力欄 | 質問入力 |
| S02-05 | 送信ボタン | 手動相談送信 |
| S02-06 | この箇所を相談 | 文脈付き相談の実行 |
| S02-07 | 深掘り | 直前回答への追加質問 |
| S02-08 | 参照文脈サマリ | 今回参照する文脈の概要 |
| S02-09 | 最新アドバイスカード | 自動助言の最新内容 |
| S02-10 | クールダウン表示 | 次回自動助言までの待機状態 |
| S02-11 | 常時モード一時停止 | 常時助言の一時停止 |
| S02-12 | 詳細を見る | S03 へ遷移 |
| S02-13 | 送信範囲確認 | S04 へ遷移 |
| S02-14 | ナレッジ管理 | S05 へ遷移 |
| S02-15 | 設定 | S06 へ遷移 |

### 5.3 S03 アドバイス詳細画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S03-01 | 助言本文 | AI 助言全文 |
| S03-02 | 根拠一覧 | ファイル、診断、差分、関連シンボル等 |
| S03-03 | 参照ファイルリンク | 該当ファイルへの移動 |
| S03-04 | 診断情報 | エラー / 警告の要約 |
| S03-05 | 変更文脈 | 最近の編集範囲の要約 |
| S03-06 | 推測表示 | 推測を含む助言の明示 |
| S03-07 | 深掘り質問 | 追加質問 |
| S03-08 | ナレッジ候補化 | ナレッジ保存対象化 |
| S03-09 | 戻る | S02 へ戻る |

### 5.4 S04 送信範囲確認画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S04-01 | 送信対象カテゴリ一覧 | 送信対象カテゴリの一覧 |
| S04-02 | 対象ファイル一覧 | 対象候補ファイルの一覧 |
| S04-03 | 除外設定一覧 | 除外条件の一覧 |
| S04-04 | 推定送信量 | 送信サイズ目安 |
| S04-05 | 除外設定を編集 | S06 へ遷移 |
| S04-06 | 閉じる | 元画面へ戻る |

### 5.5 S05 ナレッジ管理画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S05-01 | 検索欄 | ナレッジ検索 |
| S05-02 | 状態フィルタ | 有効 / 無効 / すべて |
| S05-03 | ナレッジ一覧 | 要約、状態、更新日時の表示 |
| S05-04 | ナレッジ詳細 | 本文やメモの表示 |
| S05-05 | 編集 | ナレッジ編集 |
| S05-06 | 無効化 | 再利用対象から外す |
| S05-07 | 削除 | ナレッジ削除 |
| S05-08 | エクスポート | JSON または Markdown 出力 |
| S05-09 | リセット | 全件リセット |
| S05-10 | 戻る | 元画面へ戻る |

### 5.6 S06 設定画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S06-01 | 初期モード | 起動時のモード設定 |
| S06-02 | 常時モードON/OFF | 常時モード利用可否 |
| S06-03 | リクエスト間隔 | 自動助言の最小間隔 |
| S06-04 | アイドル時間 | 自動助言前の必要無操作時間 |
| S06-05 | 重複抑制 | 類似助言の繰り返し防止 |
| S06-06 | 送信対象設定 | 各文脈要素の有効 / 無効 |
| S06-07 | 除外glob設定 | ユーザー指定除外パターン |
| S06-08 | 大容量閾値 | 除外ファイルサイズ上限 |
| S06-09 | ナレッジ再利用ON/OFF | ナレッジ利用の有効 / 無効 |
| S06-10 | 送信前確認 | 毎回確認表示の有無 |
| S06-11 | 保存 | 設定保存 |
| S06-12 | 初期値に戻す | 設定初期化 |
| S06-13 | 戻る | 元画面へ戻る |

### 5.7 S07 制限・エラー画面

| 項目ID | 項目名 | 内容 |
|---|---|---|
| S07-01 | 状態タイトル | 同意未取得などの状態名 |
| S07-02 | 状態説明 | 問題の説明 |
| S07-03 | 推奨アクション | 次に行うべき操作 |
| S07-04 | 再試行 | 接続 / 送信再試行 |
| S07-05 | Copilotセットアップ案内 | Copilot 導入案内 |
| S07-06 | Workspace信頼案内 | Workspace Trust への誘導 |
| S07-07 | 設定を開く | S06 へ遷移 |
| S07-08 | 戻る | 元画面または S01 へ戻る |
