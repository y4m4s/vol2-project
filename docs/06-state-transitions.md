# 06. 状態遷移

## 状態遷移図

```mermaid
stateDiagram-v2
    [*] --> 未接続

    未接続 --> 接続中: Copilotに接続
    接続中 --> 利用不可: Copilot未利用可 / untrusted workspace / offline
    接続中 --> 同意待ち: モデル取得成功
    同意待ち --> 接続済み_必要時: 同意完了 + 初回リクエスト成功
    同意待ち --> 未接続: 同意キャンセル

    接続済み_必要時 --> 応答生成中: 手動相談
    応答生成中 --> 接続済み_必要時: 応答完了

    接続済み_必要時 --> 接続済み_常時待機: 常時モードON
    接続済み_常時待機 --> 応答生成中: トリガー発火
    応答生成中 --> 接続済み_常時待機: 応答完了
    接続済み_常時待機 --> 接続済み_必要時: 常時モードOFF

    接続済み_必要時 --> 制限中: quota超過 / offline / 一時エラー
    接続済み_常時待機 --> 制限中: quota超過 / offline / 一時エラー
    制限中 --> 接続済み_必要時: 再試行成功 / 復旧

    利用不可 --> 未接続: 条件改善
    接続済み_必要時 --> 未接続: 権限取り消し
    接続済み_常時待機 --> 未接続: 権限取り消し
```

## 画面と状態の対応

| 状態 | 表示画面 |
|---|---|
| 未接続 | S01 初回接続画面 |
| 接続中 / 同意待ち | S01 のローディング / 確認状態 |
| 接続済み_必要時 | S02 メイン画面 |
| 接続済み_常時待機 | S02 メイン画面 |
| 応答生成中 | S02 上でストリーミング表示 |
| 制限中 / 利用不可 | S07 制限・エラー画面 |
